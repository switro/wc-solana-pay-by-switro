## Installation

1. Download the plugin zip file.  
2. Upload it to your WordPress site via **Plugins ‚Üí Add New ‚Üí Upload Plugin**.  
3. Activate the plugin through the **Plugins** menu in WordPress.  
4. Proceed to the configuration steps below.

---

## Configuration Steps

1. **Get your API Key**  
   - Login to your [Switro account](https://www.switro.com/auth/login) ‚Üí **API Settings** ‚Üí **Copy API Key**.  
2. **Configure in WordPress**  
   - Go to **WooCommerce ‚Üí Settings ‚Üí Payments ‚Üí Switro Solana Wallet**.  
   - Paste the copied API Key into the API Key field and save changes.  
3. **Set up Webhooks**  
   - From the WordPress admin ‚Üí **Payments ‚Üí Switro Solana Wallet**, copy the **Webhook URL** provided.  
   - Go back to **Switro Dashboard ‚Üí API Settings** ‚Üí paste the Webhook URL ‚Üí **Save Webhook**.  
4. Your store is now ready to accept Solana payments!
